## College Student Attendance System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/2b4a3ad426a54f558c21721be32dad97/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Teacher, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Announcement Management
6. Grade Management
7. Student Attendance Management
8. Student Leave Management



**Teacher Functions:**
1. Login
2. Change Password
3. Personal Information
4. Announcement Management
5. Class Management
6. Grade Management
7. Student Attendance Management
8. Student Leave Management
9. Student Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Basic Data Management (Announcement Type Management, Subject Management, Grade Management, Student Attendance Type Management, Leave Type Management)
5. Announcement Management
6. Class Management
7. Grade Management
8. Student Attendance Management
9. Student Attendance Details Management
10. Student Leave Management
11. Teacher Management
12. Student Management
